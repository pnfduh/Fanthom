# Installation

1. Download this repository as a ZIP and extract it
2. Right-click `Fathom Installer.bat`
3. Select **Run as administrator**
4. Click **Yes** on the UAC prompt
5. Wait for the installer to complete

That's it. The agent will start automatically and run in the background at every login.

---

**To uninstall:**
Right-click `Fathom Uninstaller.bat` and select **Run as administrator**

---

**Requirements:**
- Windows 10 or 11
- Internet connection (Python will be installed automatically if needed)
